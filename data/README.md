# Data Folder

This folder contains all datasets used in the data analysis project.

## Subfolders

- **raw/** - Original, immutable datasets. Do not modify files in this folder.
- **processed/** - Cleaned and transformed datasets ready for analysis.
- **external/** - Datasets from external sources or third-party APIs.

## Guidelines

1. Always keep raw data intact
2. Document the source of all external data
3. Use processed folder for cleaned versions
4. Add large files to .gitignore

## Supported Formats

- CSV files
- Excel files (.xlsx, .xls)
- Parquet files
- JSON files
